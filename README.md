## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger Api documentation

See the documentation of the rest api, visit the following url, after having built the project: [link to open](http://localhost:3000/documentation#/).
